# Summarize single-cell TIRTLseq data

**\[experimental\]** Returns a summary of single-cell TIRTLseq data,
printing the total number of wells with data, number of wells missing an
alpha/beta chain, number of wells with a second alpha chain, etc.

## Usage

``` r
summarize_scTIRTLseq(df)
```

## Arguments

- df:

  a data frame with single-cell TIRTLseq data
