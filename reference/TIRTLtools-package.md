# TIRTLtools: Analysis Tools for TIRTLseq Data

A suite of tools for analyzing paired T-cell receptor (TCR) datasets
created using the TIRTL-seq assay. Contains an efficient GPU
implementation of TCRdist (Dash et al., Nature 2017).
