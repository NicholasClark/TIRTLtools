# Remove TCRs with nonfunctional CDR3 amino acid sequences

**\[experimental\]**

## Usage

``` r
filter_nonfunctional_TCRs(df, verbose = TRUE)
```

## Arguments

- df:

  data frame with paired TCRs

- verbose:

  whether to print number of TCRs removed
